# 无线通信技术实验报告：分集与扩频通信

## 1. 实验目的

本实验围绕无线通信中的两类抗衰落、抗干扰技术展开：分集合并与直接序列扩频通信。实验目标包括：

1. 理解瑞利平坦衰落信道中深衰落导致误码率升高的机制。
2. 实现选择合并（Selection Combining, SC）和最大比合并（Maximal Ratio Combining, MRC），比较单分支、SC 和 MRC 在不同 SNR 下的 BER 表现。
3. 掌握 m 序列的线性反馈移位寄存器生成方法，以及 DSSS 中扩频、解扩和处理增益的基本计算。
4. 通过仿真观察 DSSS 在窄带干扰下的误码率改善，但不把有限仿真结果过度外推为真实系统性能结论。
5. 使用 pytest、pylint 和自动评分脚本检查代码正确性、结果文件完整性和代码质量。

## 2. 实验原理

### 2.1 分集合并

无线信道中的多径传播会使不同传播路径的信号在接收端相加。当多个路径相位关系不利时，接收信号可能显著衰减，形成深衰落。瑞利平坦衰落常用于描述无直射径或散射占主导时的复信道系数，本实验采用的接收模型为：

```text
r_i[n] = h_i[n] x[n] + w_i[n]
```

其中 `x[n]` 是 BPSK 发送符号，`h_i[n]` 是第 `i` 个分支的复衰落系数，`w_i[n]` 是复高斯白噪声。单分支接收在某个符号处若 `|h|` 很小，即使平均 SNR 较高，均衡后的噪声也会被放大，导致误判。

选择合并 SC 的思想是每个符号只选择瞬时信道功率最大的分支：

```text
k = argmax_i |h_i[n]|^2
y[n] = r_k[n] / h_k[n]
```

SC 的优点是实现简单，只需要从多个分支中选择一个最强分支；缺点是没有利用其他非最强分支中的有效信号能量。

最大比合并 MRC 使用所有分支。它先用信道共轭校正相位，再按信道幅度进行加权，并用总信道功率归一化：

```text
y[n] = sum_i conj(h_i[n]) r_i[n] / sum_i |h_i[n]|^2
```

在独立 AWGN 且信道估计准确的前提下，MRC 能最大化合并后的瞬时 SNR，因此通常优于 SC。该结论依赖上述模型条件，若分支噪声相关、信道估计偏差明显或硬件分支不独立，优势可能减弱。

### 2.2 扩频通信

直接序列扩频（DSSS）先将比特映射为 BPSK 符号，再用高速伪随机 PN 码片序列扩展每个符号。若 PN 长度为 `N`，则每个数据符号被扩展为 `N` 个码片：

```text
c[nN + k] = a[n] p[k]
```

接收端使用同一 PN 序列进行相关解扩：

```text
z[n] = sum_k r[nN + k] p[k]
```

若相关输出非负，判决为 bit 0；若为负，判决为 bit 1。理想同步条件下，期望信号在解扩时相干累加，而未同步的窄带干扰会被扩展到更宽频带内，等效干扰功率在判决统计量中被摊薄。

m 序列由线性反馈移位寄存器产生。本实验约定寄存器从左到右书写，抽头位置为从左到右的 1-based 索引，每拍输出右端寄存器，抽头异或作为反馈插入左端。输出比特再映射为双极性码片：`0 -> +1`，`1 -> -1`。扩频因子为 `N` 时，理想处理增益为：

```text
G_p = 10 log10(N) dB
```

本实验使用长度为 31 的 PN 序列，因此处理增益约为 `14.91 dB`。

## 4. 实验方法

### 4.1 Part 1：分集合并

首先生成随机 0/1 比特，并使用 BPSK 映射：

```text
0 -> +1
1 -> -1
```

随后使用 `rayleigh_fading_branches` 生成多分支瑞利平坦衰落接收信号。每个分支包含独立复信道系数和复 AWGN。对每个 SNR 点分别计算三种接收方式的 BER：

1. 单分支：只使用第一个接收分支，直接做 `received[0] / channel[0]` 均衡。
2. SC：对每个符号选择 `|h|^2` 最大的分支，再做 `r / h`。
3. MRC：实现 `sum(conj(h) * r) / sum(|h|^2)`，使用全部分支的加权合并结果。

仿真参数为 SNR = `[0, 3, 6, 9, 12, 15] dB`，比特数为 6000，分支数为 2，随机种子为 2026。最后绘制 BER 曲线和部分波形快照。

### 4.2 Part 2：DSSS 扩频通信

首先使用 LFSR 生成 PN 码片序列。实验中使用初始状态 `[1, 1, 1, 0, 1]`，抽头 `[5, 2]`，长度为 31。然后对随机比特进行 BPSK 映射，并让每个符号乘完整 PN 序列，得到扩频码片流。

在对比实验中，未扩频 BPSK 和 DSSS 码片流都加入同样形式的窄带余弦干扰，再加入 AWGN。DSSS 接收端按扩频因子 reshape 后与同一 PN 序列相关，依据相关输出符号进行硬判决。仿真参数为 SNR = `[-6, -3, 0, 3, 6, 9] dB`，比特数为 3000，PN 长度为 31。

## 6. 结果分析

瑞利衰落会造成深衰落，是因为接收端多条散射路径的复幅度随机叠加。当相位相消时，瞬时信道幅度 `|h|` 可能远低于平均水平。此时均衡操作 `r / h` 会同时放大噪声，导致 BER 出现明显恶化。因此，只提高平均 SNR 并不能完全消除深衰落带来的误码风险。

SC 与 MRC 的根本区别在于是否利用所有分支。SC 每个符号只取瞬时最强分支，降低了所有分支同时深衰落的概率，但丢弃了其他分支的信号能量。MRC 对所有分支做相位校正和幅度加权，理论上能在独立噪声、准确 CSI 条件下最大化输出 SNR。本实验中同一 SNR 下 BER 基本满足 `MRC <= SC <= 单分支`，与理论预期一致。

从 Part 1 的数值看，0 dB 时单分支 BER 为 0.1482，SC 降至 0.0752，MRC 降至 0.0570；15 dB 时 MRC 在 6000 bit 仿真中没有观测到误码。这里不能解释为真实 BER 等于 0，只能说明在当前有限样本和固定随机种子下未出现错误。若要给出更稳健的高 SNR BER 估计，需要增加比特数或使用置信区间。

DSSS 的处理增益由扩频因子决定。本实验 PN 长度为 31，对应 `10 log10(31) = 14.91 dB`。扩频后，期望信号与本地 PN 序列同步，解扩相关时能够相干累加；窄带干扰与 PN 序列不匹配，在相关过程中不会按同一符号方向累加，因此对判决统计量的影响被削弱。Part 2 的结果显示，在相同窄带干扰设置下，DSSS BER 显著低于未扩频 BER。

需要指出，本实验中的 DSSS 结果依赖干扰幅度、干扰频率、PN 长度、同步完全正确等条件。若存在码片同步偏差、频偏、强多径或 PN 序列相关性不足，实际性能会下降。因此，本实验结果可以支持“DSSS 在本仿真条件下显著抑制窄带干扰”，但不足以单独证明其在任意无线场景下都能达到同等增益。

## 7. 实验心得

本实验的关键不是把公式写进代码，而是保证代码实现与信道模型和判决逻辑严格一致。MRC 如果漏掉共轭或归一化，输出在复信道下会出现系统性错误；SC 如果只按分支平均功率选择，而不是逐符号选择 `|h|^2` 最大分支，也会偏离定义。DSSS 部分同样如此，PN 码片的双极性映射、LFSR 抽头方向、reshape 维度和相关判决符号都会直接影响结果。

从实验结果看，分集和扩频都能改善误码表现，但它们解决的问题不同。分集主要缓解随机衰落中的深衰落风险，DSSS 主要通过带宽换处理增益来抑制窄带干扰。二者都不是无条件提升性能的“通用结论”，其有效性依赖信道独立性、同步、信道估计和噪声干扰模型。报告中应避免把有限仿真曲线写成现实系统的绝对性能证明。

自动测试对本实验有实际价值。pytest 能检查公式实现、输入合法性和无噪声恢复能力；pylint 能暴露导入、异常捕获、复杂度和格式问题。AI 辅助可以提高实现效率，但最终可信性来自可复现测试、结果文件和对公式边界条件的理解。
