# 分集通信理论速览

## 1. 为什么需要分集

无线链路中，接收信号会受到多径传播、遮挡和移动的影响。若多个路径相消，接收端会出现短时间的深衰落。即使平均 SNR 不低，深衰落也会让误码率突然升高。

分集的核心思想是：让接收端获得多个统计上尽量独立的信号副本。只要不是所有副本同时处于深衰落，就有机会恢复发送符号。

## 2. 瑞利平坦衰落模型

本实验采用每个符号独立的平坦瑞利衰落模型：

```text
r_i[n] = h_i[n] x[n] + w_i[n]
```

其中 `i` 是分支编号，`h_i[n]` 是复高斯衰落系数，`w_i[n]` 是复 AWGN。

## 3. 选择合并 SC

选择合并每个符号只使用瞬时信道最强的分支：

```text
k = argmax_i |h_i[n]|^2
y[n] = r_k[n] / h_k[n]
```

SC 实现简单，但没有利用所有分支的信息。

## 4. 最大比合并 MRC

最大比合并先用信道共轭校正每个分支的相位，再按信道幅度加权：

```text
y[n] = sum_i conj(h_i[n]) r_i[n] / sum_i |h_i[n]|^2
```

在 AWGN 条件下，MRC 可以最大化合并后的瞬时 SNR，通常优于 SC。

## 5. 观察重点

- 单分支在瑞利衰落下 BER 下降较慢。
- SC 能降低深衰落概率。
- MRC 通常在同样分支数下获得更低 BER。
- 分支数越多，分集阶数越高，但硬件和信道估计成本也越高。
